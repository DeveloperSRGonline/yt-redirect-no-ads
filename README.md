# yt-redirect-no-ads

A custom Chrome extension to watch YouTube videos without ads.

## How to Install in Chrome

Follow these steps to add this custom extension to your browser:

1. **Download the Code:** Clone this repository or download it as a ZIP file and extract it to a folder.
2. **Open Extensions Page:** Open Google Chrome and type `chrome://extensions/` in the address bar.
3. **Enable Developer Mode:** Turn on the **Developer mode** toggle switch located in the top right corner.
4. **Load Extension:** Click the **Load unpacked** button that appears in the top left.
5. **Select Folder:** Select the folder containing your extension files.

The extension is now installed and active!
